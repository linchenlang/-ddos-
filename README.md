# DDOS攻击与防御学习工具集

## 📝 项目简介

本项目是一个用于**网络安全教育**和**防御技术研究**的工具集合，主要演示TCP Flood（SYN Flood）攻击的基本原理以及相应的防御策略。通过模拟攻击场景，帮助安全研究人员、网络管理员和学生学习如何在受控环境中识别、分析和缓解DDoS攻击。

## 🛠️ 包含的工具

### 1. TCP Flood攻击演示脚本 (`async_tcp_flood.py`)
- **功能**：模拟TCP洪水攻击，演示DDoS攻击的基本原理
- **特点**：
  - 支持多种攻击模式（快速、慢速、正常）
  - 可调节的并发连接数和攻击频率
  - 详细的统计信息和实时监控
  - 连接池和资源管理，避免系统过载

### 2. 简易HTTP靶机服务器 (`simple_http_server.py`)
- **功能**：一个简单的HTTP服务器，用于作为攻击目标
- **特点**：
  - 记录所有传入请求的详细信息
  - 显示实时统计数据和性能指标
  - 帮助分析攻击特征和模式

### 3. 系统限制优化脚本 (`permanent_limit_fixer.py`)
- **功能**：提高Linux系统资源限制，解决`errno 24 (EMFILE)`错误
- **特点**：
  - 永久提高文件描述符和进程数限制
  - 优化网络连接相关参数
  - 自动备份原始配置，确保安全

## 🎯 学习目标

通过使用本项目，您可以学习到：

### 攻击面理解
1. **TCP洪水攻击原理**：了解SYN Flood、ACK Flood等攻击方式
2. **资源耗尽攻击**：理解如何通过大量连接耗尽系统资源
3. **攻击特征分析**：识别攻击流量的模式和特征

### 防御技术掌握
1. **系统加固**：学习如何配置系统以抵御DDoS攻击
2. **监控与检测**：建立有效的攻击检测机制
3. **缓解策略**：实施流量清洗、限速和黑名单等防御措施
4. **应急响应**：制定针对DDoS攻击的应急响应计划

## 🚀 快速开始

### 环境要求
- Python 3.6+
- Linux系统（推荐Kali Linux或Ubuntu）
- 隔离的测试网络环境

### 基础测试步骤

# 1. 启动靶机服务器
python3 simple_http_server.py

# 2. 在新终端中运行攻击演示（针对本地靶机）
python3 tcp_flood.py --target 127.0.0.1 --port 8080 --threads 50 --duration 30

# 3. 观察攻击效果和服务器响应


### 高级使用

# 使用不同攻击模式
python3 tcp_flood.py --mode fast     # 快速攻击模式
python3 tcp_flood.py --mode slowloris # 慢速攻击模式

# 调整攻击参数
python3 tcp_flood.py --threads 200 --duration 60 --rate 50


## 🛡️ 防御建议

### 基础设施层面
1. **系统加固**：
   - 调整系统资源限制（使用本项目提供的优化脚本）
   - 启用SYN Cookie保护
   - 配置合理的连接超时时间

2. **网络防护**：
   - 部署Web应用防火墙（WAF）
   - 使用CDN服务分散流量
   - 配置DDoS防护设备或服务

3. **监控告警**：
   - 实施流量基线监控
   - 设置异常流量告警
   - 定期分析访问日志

### 应用层面
1. **请求限制**：
   - 实施IP频率限制
   - 验证用户身份和会话
   - 使用验证码和挑战响应机制

2. **架构设计**：
   - 采用分布式架构
   - 实现负载均衡
   - 设计弹性伸缩方案
