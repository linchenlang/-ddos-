**如何合法使用**



**1. 建立封闭测试环境**

&nbsp;  - 使用虚拟机（VMware/VirtualBox）

&nbsp;  - 确保完全隔离的网络

&nbsp;  - 仅测试自己拥有完全控制权的系统



**2. 获取书面授权**

&nbsp;  - 如需测试他人系统，必须获得书面授权

&nbsp;  - 明确测试范围和时间

&nbsp;  - 仅理解原理，不进行实际攻击
